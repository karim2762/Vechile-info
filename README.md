## ✨ Overview

KEXER Vehicle Intelligence Toolkit is a modern vehicle registration lookup platform designed for educational and development purposes. The system retrieves vehicle details through API integration and presents them in a clean, responsive, and user-friendly interface.

Built using Python, HTML, and modern web technologies, this project demonstrates real-world API integration and vehicle data processing.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/karim2762/Vechile-info.git
```

Move into the project directory:

```bash
cd Vehicle-Toolkit
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```

---

## 🔑 Configuration

Add your API credentials inside the backend file:

```python
API_KEY = "YOUR_API_KEY"
```
